# ECG Bioinstrumentation System

## Overview

The **ECG Bioinstrumentation System** is a biomedical signal acquisition project developed to monitor and visualize the electrical activity of the human heart. The system acquires low-amplitude ECG signals, conditions them through analog circuitry, and displays the processed waveform for analysis.

This project demonstrates the principles of biomedical instrumentation, analog circuit design, signal conditioning, and physiological signal measurement.

---

## Objectives

- Acquire low-amplitude ECG signals safely.
- Amplify and filter the ECG signal.
- Remove noise and power-line interference.
- Display a clean ECG waveform for monitoring and analysis.

---

## Features

- ECG signal acquisition
- Instrumentation amplifier-based signal conditioning
- Low-pass and high-pass filtering
- Noise reduction
- Real-time waveform monitoring
- Compact hardware implementation

---

## System Architecture
